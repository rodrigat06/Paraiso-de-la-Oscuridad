# Medios del proyecto

Los artistas oficiales y los artistas añadidos pueden usar imágenes y audios cargados desde la propia aplicación.

- Para artistas nuevos, sube una portada al crearlos desde el botón Añadir.
- En la página del artista puedes añadir álbumes, EPs y singles con portada propia.
- Los singles aceptan audio local y se reproducen como vinilos interactivos.
- No se incluyen canciones comerciales descargadas automáticamente para evitar problemas de copyright. Sustituye los archivos cuando tengas permisos o material propio.
