Como esta organizada la pagina

Cada carpeta representa una parte visible de la web. Dentro tiene su HTML, CSS, JS e imagenes si hacen falta.

inicio

Portada principal. Presenta la idea y manda a explorar, guia o login.

explorar

Listado de artistas. Tiene buscador funcional y enseña propuestas guardadas.

artistas

Una carpeta por cantante. Cada ficha incluye biografia, albumes, singles, imagenes y audios.

login / registro

Formularios separados y simples. Cada uno llama a su endpoint del servidor.

cantantes

Permite añadir artistas famosos de una lista cerrada para mantener el genero de la web.

sin admin ni significados

Se quitaron porque complicaban la pagina y no eran necesarios para la idea actual.
