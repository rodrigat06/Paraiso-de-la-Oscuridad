# Carpeta de Jazmin Bean

Todo lo especifico de Jazmin Bean vive aqui.

```text
Albums/        HTML de albumes, EPs y singles.
audio/         Audios usados por sus paginas.
css/           Estilos propios de sus paginas.
data/          Datos de significados.
gifs/          GIFs usados en navegacion y tarjetas.
img/           Imagenes y portadas.
js/            JavaScript propio de sus paginas.
videos/        Videos usados por sus paginas.
significado.html  Plantilla unica para significados.
```

Las URLs antiguas de `significados-canciones/*.html` siguen funcionando gracias a `JazminMeaningController.java`.
