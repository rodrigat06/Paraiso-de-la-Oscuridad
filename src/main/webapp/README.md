# Paraiso de la Oscuridad

Estructura simple para encontrar cada cosa sin abrir veinte archivos iguales:

- `pagina-principal.html`: portada de la web.
- `inicio/pagina-principal.css`: estilos de la portada.
- `inicio/pagina-principal.js`: JavaScript minimo de la portada.
- `explorar/explorar.html`: pantalla para buscar artistas.
- `explorar/explorar.css`: estilos del buscador de artistas.
- `explorar/explorar.js`: logica del buscador de artistas.
- `administrador/administrador.html`: pantalla privada para crear o editar artistas.
- `administrador/administrador.css`: estilos del administrador.
- `administrador/administrador.js`: logica del administrador.
- `guia/guia.html`: explicacion de como funciona la web.
- `guia/guia.css`: estilos de la guia.
- `login/login.html`: pantalla de inicio de sesion.
- `login/login.css`: estilos del login.
- `login/login.js`: logica del login.
- `registro/registro.html`: pantalla de registro.
- `registro/registro.css`: estilos del registro.
- `registro/registro.js`: logica del registro.
- `recuperar/recuperar.html`: pantalla para recuperar contraseña.
- `recuperar/recuperar.css`: estilos de recuperar contraseña.
- `recuperar/recuperar.js`: logica de recuperar contraseña.
- `artistas/artist-store.js`: datos base de artistas, albumes, EPs, singles y canciones.
- `artistas/media-store.js`: guarda imagenes y audios elegidos desde el ordenador.
- `artistas/ficha/ficha-artista.html`: ficha dinamica para artistas nuevos.
- `artistas/ficha/ficha-artista.js`: logica de la ficha dinamica.
- `artistas/jazmin-bean/jazmin-bean.html`: ficha de Jazmin Bean.
- `artistas/jazmin-bean/jazmin-bean.css`: estilos de Jazmin Bean.
- `artistas/jazmin-bean/jazmin-bean.js`: logica de Jazmin Bean.
- `artistas/melanie-martinez/melanie-martinez.html`: ficha de Melanie Martinez.
- `artistas/melanie-martinez/melanie-martinez.css`: estilos de Melanie Martinez.
- `artistas/melanie-martinez/melanie-martinez.js`: logica de Melanie Martinez.
