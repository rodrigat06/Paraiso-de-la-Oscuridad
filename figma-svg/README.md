# SVG separados para Figma

Ahora los SVG muestran la pagina completa, incluyendo lo que aparece al hacer scroll.

Carpetas:

- `ordenador/`: pantallas completas en formato ordenador.
- `tablet/`: pantallas completas en formato tablet.
- `telefono/`: pantallas completas en formato telefono.

En cada carpeta hay un SVG por pantalla y un resumen llamado `todas-las-pantallas-*.svg`.
