# SVG separados para Figma

Carpetas:

- `ordenador/`: pantallas en formato ordenador.
- `tablet/`: pantallas en formato tablet.
- `telefono/`: pantallas en formato telefono.

En cada carpeta hay un SVG por pantalla y un resumen llamado `todas-las-pantallas-*.svg`.

Estos SVG salen de capturas reales de la pagina actual, asi que sirven como boceto fiel para importar en Figma.
