# SVG para Figma

Estos archivos son bocetos responsive de la pagina actual, generados desde capturas reales del navegador.

Archivo principal para importar en Figma:

- `paraiso-responsive-figma.svg`

Ese SVG contiene las pantallas principales en tres tamaños:

- ordenador
- tablet
- movil

Archivos separados por pantalla:

- `login-responsive.svg`
- `portada-responsive.svg`
- `explorar-responsive.svg`
- `jazmin-responsive.svg`
- `melanie-responsive.svg`
- `administrador-responsive.svg`

Cada SVG se puede arrastrar directamente a Figma. Si Figma lo importa como una sola imagen, sigue sirviendo como maqueta visual exacta. Si necesitas editar cada elemento como vector real, habria que reconstruir los componentes manualmente en Figma encima de estas referencias.
